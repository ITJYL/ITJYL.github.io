# 01. 기능 명세 (MVP)

요구사항 명세서(REQ Specification v1.0) 기준으로 1차 MVP 범위와 고도화 기능을 정리합니다.

---

## 우선순위 기준

| 우선순위 | 의미 | 개발 단계 |
|----------|------|-----------|
| Critical | MVP 출시 전 반드시 | Phase 1 |
| High | 추가 요청, 가능하면 포함 | Phase 1.5 |
| Medium | 고도화 | Phase 2 |
| Low | 추후 검토 | Phase 3 |

---

## Phase 1 — MVP 필수 (Critical)

### REQ-001 회원 관리

| 기능 | 상세 | 비고 |
|------|------|------|
| REQ-001-01 이메일 회원가입 | 이메일 중복 확인, 이메일 인증(6자리 코드, 10분 유효), 비밀번호 암호화(BCrypt) | 구현 완료 |
| REQ-001-02 휴대폰 인증 가입 | SMS 인증번호 발송, 3분 유효, 번호 중복 확인 | UI 탭만 |
| REQ-001-03 SNS 간편 로그인 | Google OAuth 2.0, 소셜 계정 정보 연동 | 버튼만 |
| REQ-001-04 로그인 보안 | 새 기기 로그인 알림, 세션 관리(자동 로그아웃) | JWT Stateless |

### REQ-002 구매 요청

| 기능 | 상세 |
|------|------|
| 상품 URL 입력 | 다중 URL, 유효성 검증, 러시아 쇼핑몰(Wildberries, Ozon 등) |
| 상품 사진 첨부 | 최대 10장, JPG/PNG/WEBP, 개당 5MB, 자동 리사이징 |
| 구매 요청서 작성 | 상품명, 수량, 옵션(색상/사이즈), 메모 |

### REQ-003 1:1 문의 게시판

| 기능 | 상세 |
|------|------|
| 문의 등록 | 제목·내용, 파일 첨부(최대 5개, 10MB), 카테고리(주문/배송/결제/기타) |
| 문의 조회 | 본인 문의만 조회, 답변 상태(대기중/답변완료), 작성일시 |
| 답변 알림 | 답변 등록 시 자동 알림 |

### REQ-004 공지사항

| 기능 | 상세 |
|------|------|
| 공지 조회 | 최신순 정렬, 중요 공지 상단 고정, 조회수 |
| 공지 알림 | 새 공지 등록 시 푸시 알림 |

### REQ-005 환율 자동 계산

| 기능 | 상세 |
|------|------|
| 환율 API 연동 | 외부 환율 API, 1일 1회 이상 갱신, 관리자 수동 설정 |
| 가격 계산 | 루블 → 원화 자동 변환, 수수료 계산, 예상 총액 |
| 환율 표시 | 현재 적용 환율·업데이트 시간 표시 |

### REQ-006 푸시 알림

| 기능 | 상세 |
|------|------|
| 알림 시점 | 회원가입, 로그인(새 기기), 문의 답변, 공지, 프로모션, 구매 완료 |
| 알림 채널 | 푸시(모바일), SMS, 이메일 |
| 알림 설정 | 사용자별 ON/OFF |

### REQ-007 관리자 페이지

| 기능 | 상세 |
|------|------|
| 회원 관리 | 목록(검색/필터), 상세, 상태(정상/정지/탈퇴), 통계 |
| 문의 관리 | 목록, 답변 작성/수정/삭제, 상태 관리, 답변 템플릿 |
| 공지 관리 | 작성/수정/삭제, 중요 공지 설정, 예약 발행 |
| 통계/대시보드 | 일별/월별 가입자, 문의 현황, 구매 요청, 매출 |
| 환율 설정 | 수동 환율 입력, 히스토리, 수수료율 설정 |
| 권한 관리 | 복수 관리자, 역할 분리(슈퍼 관리자/상담사) |

## Phase 1.5 — 추가 요청 (High)

### REQ-016 잔액 표시 시스템

| 기능 | 상세 |
|------|------|
| 잔액 충전 | 계좌이체 충전, 카드 충전, 충전 내역 조회 |
| 잔액 표시 | 메인 화면 잔액, 실시간 업데이트, 부족 시 알림 |
| 사용 내역 | 충전/사용 내역 조회, 기간별 필터, 다운로드(CSV/PDF) |

### REQ-017 계좌이체 시스템

| 기능 | 상세 |
|------|------|
| 가상계좌 발급 | 사용자별 전용 계좌, 입금 자동 확인, 입금 완료 알림 |
| 이체 내역 관리 | 이체 내역 조회, 입금 확인 상태, 영수증 |
| 중고거래 대행 | 에스크로, 거래 완료 후 정산, 취소 시 환불 |

### REQ-015 구매 완료 알림

| 기능 | 상세 |
|------|------|
| 단계별 알림 | 구매 확정, 배송 시작, 배송 완료, 상태 추적 |
