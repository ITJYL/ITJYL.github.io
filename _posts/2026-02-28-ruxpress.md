# 00. 프로젝트 개요

### 목표

- 한국-러시아 간 상품 구매대행 서비스를 제공하는 모바일/웹 플랫폼 구축

### 범위

- 1차: MVP 개발
- 2차: 고도화 기능 추가

### 일정/가정 (초안)

- 개발 기간: 1차 MVP 3~4주
- 개발 인력: 4명

### 핵심 데이터 엔티티

- 회원(User)
- 구매 요청(Purchase Request)
- 문의(Inquiry)
- 공지사항(Notice)
- 알림(Notification)
- 환율(Exchange Rate)
- 포인트/쿠폰/잔액/거래 내역

### 기술 스택

**Frontend**

- [ ] 프레임워크: React
- 네이티브 앱 + 웹뷰 → 웹 (React)

**Backend**

- [ ] 언어/프레임워크: Spring Boot
- [ ] 데이터베이스: Maria DB (AWS RDS)
- [ ] 캐싱: Redis (환율 정보, 세션 관리)
- [ ] 큐: Kafka (메신저)
- [ ] 파일 스토리지: AWS S3
- [ ] 모바일 푸시: Firebase FCM
